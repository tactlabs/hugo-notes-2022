### Create a regular expression pattern that breaks apart emails


```python
pattern = '([A-Z0-9._%+-]+)@([A-Z0-9.-]+)\\.([A-Z]{2,4})'
```

### Find everything in df.email that contains that pattern


```python
df['email'].str.findall(pattern, flags=re.IGNORECASE)
```
