## Create a color palette and set it as the current color palette


```python
flatui = ["#9b59b6", "#3498db", "#95a5a6", "#e74c3c", "#34495e", "#2ecc71"]
sns.set_palette(flatui)
sns.palplot(sns.color_palette())
```


![png](seaborn_color_palettes_29_0.png)
