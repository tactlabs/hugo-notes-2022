## Preliminaries


```python
from random import choice
```

## Create List


```python
# Make a list of crew members
crew_members = ['Steve', 'Stacy', 'Miller', 'Chris', 'Bill', 'Jack']
```

## Select Random Item From List


```python
# Choose a random crew member
choice(crew_members)
```
