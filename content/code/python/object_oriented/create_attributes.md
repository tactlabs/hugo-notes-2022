## Create Simple Class


```python
class Soldier: 
    pass
```

## Create Instance Of Class


```python
jason = Soldier()
```

## Create Attribute


```python
jason.age = 36
jason.height = 124
```

## View Attributes


```python
# Display a dictionary of attributes
jason.__dict__
```




    {'age': 36, 'height': 124}
