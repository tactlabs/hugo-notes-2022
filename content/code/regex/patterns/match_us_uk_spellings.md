## Preliminaries


```python
# Load regex package
import re
```

## Create some text


```python
# Create a variable containing a text string
text = 'It\s center and not centre.'
```

## Apply regex


```python
# Find any ISBN-10 or ISBN-13 number
re.findall(r'\bcent(?:er|re)\b', text)
```
