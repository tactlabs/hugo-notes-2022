## Preliminaries


```python
# Load library
import numpy as np
```

## Create Matrix


```python
# Create matrix
matrix = np.array([[1, 4],
                   [2, 5]])
```

## Invert Matrix


```python
# Calculate inverse of matrix
np.linalg.inv(matrix)
```
