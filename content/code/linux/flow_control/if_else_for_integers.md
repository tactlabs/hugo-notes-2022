age=23
{{< /highlight >}}

## Create If Else Statement

The code below in plain English: If the variable `age` is greater than 17, then return "Adult" otherwise return "Child".

{{< highlight markdown >}}
if (( "$age" > 17 )); then
    echo "Adult"
else
    echo "Child"
fi
{{< /highlight >}}
