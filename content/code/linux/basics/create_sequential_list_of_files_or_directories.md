## Create Sequential List Of Files

{{< highlight markdown >}}
touch script_version_{1..5}.py
{{< /highlight >}}

## Create Sequential List Of Directories

{{< highlight markdown >}}
mkdir data_{2015..2018}
{{< /highlight >}}

## View Files And Directories

{{< highlight markdown >}}
ls -l
{{< /highlight >}}
