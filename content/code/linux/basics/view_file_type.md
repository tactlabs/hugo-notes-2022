The `file` command allows us to view a file's type.

## Make File

{{< highlight markdown >}}
echo "This is some text." > sales.txt
{{< /highlight >}}

## View A File's Type

{{< highlight markdown >}}
file LICENSE.txt
{{< /highlight >}}
```
sales.txt: ASCII text
```
