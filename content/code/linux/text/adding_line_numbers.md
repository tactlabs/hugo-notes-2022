## Make File With Some Content

{{< highlight markdown >}}
echo "Jan" >> sales.txt; echo "Feb" >> sales.txt; echo "Mar" >> sales.txt
{{< /highlight >}}

## Make File Contents

{{< highlight markdown >}}
cat sales.txt
{{< /highlight >}}
```
Jan
Feb
Mar
```

## Make File Contents With Line Numbers

{{< highlight markdown >}}
cat -n sales.txt
{{< /highlight >}}
```
     1	Jan
     2	Feb
     3	Mar
```
