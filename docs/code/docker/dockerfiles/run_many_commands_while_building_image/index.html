    Run Many Commands While Building Image

    22 Jul 2020

    Create Dockerfile

    A new image layer is created every time RUN is executed, therefore it is most often advisable to chain build steps using &&.

    # Build from base image
FROM ubuntu:latest

# Run many commands
RUN apt-get update \
    && apt-get install -y git \
    && touch config.txt \
    && mkdir secret_project \
    && touch secret_project/project_alpha.txt \
    && touch secret_project/project_beta.txt \
    && touch secret_project/project_delta.txt \
    && touch secret_project/project_omega.txt \

    Build Image From Dockerfile

    Build the Dockerfile (docker build) in the current directory (.) and call the image chrisalbon/ubuntu:many-commands (--tag chrisalbon/ubuntu:many-commands).

    docker build --tag chrisalbon/ubuntu:many-commands .
